**1.웹사이트 기능:**

1. 사용자는 회원가입,로그인,ID/PW 찾기를 할수있다.
2. 사용자는 마커를 찍을수있고, 마커를 통해서 채팅방에 입장할수있다.
3. 사용자는 당근마켓 같은 서비스를 이용할수있다.
4. 사용자는 채팅방에 입장해서 실시간으로 다른 사용자와 채팅할수있다.

**2.개발환경**

1. 개발 환경: node.js,mySql
2. 배포 환경: webpack 번들링
3. 배포인프라:aws-ec2(우분투),
4. 기타: web socket.io
5. api: xmlHttpReqeust,fetch
